# 📅 Calendars — Sports Event Calendars

> **English** | [Português do Brasil](#-calendars--calendários-de-eventos-esportivos)

---

## 🏟️ About

**Calendars** is a collection of public sports event calendars in `.ics` format, designed to help fans easily track schedules and fixtures across different competitions and tournaments.
The goal is to provide standardized, and easy-to-subscribe calendar files with accurate schedules and timezone support.

## ✨ Features

- `.ics` calendar files
- Compatible with major calendar apps
- Timezone-aware events
- Organized by sport > tournament > season
- Public and easy to subscribe to
- Frequently updated schedules

## 📁 Structure

```
calendars/
└── football/       # Football event calendars
    └──── world-cup/       # FIFA World Cup event calendars
```

Each folder contains calendar data for a specific competition. More sports and tournaments will be added over time.

## 🚀 Getting Started

#### Import manually

Download the desired .ics file
Import it into your preferred calendar application

#### Subscribe via URL

You can also subscribe directly using the GitHub Pages URL so events stay updated automatically.

⚠️ Disclaimer

Schedules may change due to official event updates, delays, or organizational changes.
Always verify important match times with official organizers.

# 📅 Calendars — Calendários de Eventos Esportivos

> [English](#-calendars--sports-event-calendars) | **Português do Brasil**

---

## 🏟️ Sobre

**Calendários** é uma coleção de calendários públicos de eventos esportivos no formato `.ics`, projetada para ajudar os fãs a acompanhar facilmente os horários e jogos de diferentes competições e torneios.
O objetivo é fornecer arquivos de calendário padronizados e fáceis de assinar, com horários precisos e suporte a fusos horários.

## ✨ Funcionalidades

- Arquivos de calendário `.ics`
- Compatível com os principais aplicativos de calendário
- Eventos com suporte a fuso horário
- Organização por esporte > torneio > temporada
- Público e fácil de assinar
- Atualizações frequentes de calendário

## 📁 Estrutura

```
calendars/
└── football/       # calendário de eventos de futebol
    └──── world-cup/       # calendário de eventos da Copa do Mundo FIFA
```

Cada pasta contém os dados de calendário de uma competição específica. Mais esportes e torneios serão adicionados ao longo do tempo.
